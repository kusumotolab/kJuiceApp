items

Get item list

商品リストを取得する

query Parameters
category
string (ItemCategory)
Enum: "juice" "food"

指定されたカテゴリの商品のみ取得する．未指定の場合すべての商品を取得する．

active
boolean

activeが一致する商品のみ取得する．未指定の場合すべての商品を取得する.

Responses

local

http://localhost/backend/items

Response samples

  • 200
  • 400
Content type
application/json

    • "id": "gogotea",
    • "name": "午後の紅茶",
    • "sellingPrice": 0,
    • "costPrice": 0,
    • "category": "juice",
    • "active": true
    }
]

string (MemberAttribute)

active
boolean
